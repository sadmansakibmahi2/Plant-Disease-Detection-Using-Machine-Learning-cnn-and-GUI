To install and run a Streamlit app (`app.py`) in Python, follow these steps:

---

### ✅ Step 1: Install Streamlit

Open a terminal (Command Prompt, PowerShell, or Terminal) and run:

```bash
pip install streamlit
```

> Make sure you're in your virtual environment if you're using one (like `venv`, `conda`, etc.).

---

### ✅ Step 2: Check Installation (Optional)

After installation, verify that Streamlit is installed correctly:

```bash
streamlit hello
```

This runs a sample app in your browser. If it works, you're good to go.

---

### ✅ Step 3: Run Your `app.py`

Navigate to the directory where your `app.py` file is located. Then run:

```bash
python -m streamlit run app.py
```

This will launch the app in your default web browser (usually at `http://localhost:8501`).
